# Доступные команды

В данном документе представленны все комманды для [Telegram бота][telegram bot url]. 
Данные команды работают исключительно в приватном чате с ботом.


## Управление группами

- /addgroup [наименование группы] - Добавление новой группы
- /getgroupinfo [id группы] - Получение информации о группе
- /getgroupsinfo - Получение информации о всех группах, в которых состоит пользователь

## Управление пользователями

- /adduser [id роль] [telegram id пользователя] [Имя] - Добавление пользователя
- /addusertogroup [id группы] [telegram id пользователя] - Добавление пользователя в группу
- /changeuser [telegram id пользователя] [id роль] - Изменение пользователя

## Управление чатами

- /addchattogroup [id чата] [id группы] - Добавление пользователя в группу для рассылки
- /getchatsinfo - Получение информации о всех чатах, которые есть в БД. При вызове данной команды проверяются все группы на наличие [Telegram бота][telegram bot url]
- /leavechatbyid [telegram chat id] - Команда для выхода бота из чата
- /removechatfromgroup [id чата] [id группы] - Удаление чата из группы для рассылки

## Рассылка сообщений

- /sendallchatbygroup [id группы] [текст сообщения] - Отправка сообщения во все чаты, которые имееются в группе
- /sendallchat [текст сообщения] - Отправка сообщения во все чаты
- /sendchatbyid [chat id telegram] [текст сообщения] - Отправка сообщения в чат по идентификатору Telegram

## Утилитарные команды
 
- /getmyid - Получение идентификатора учетной записи Telegram пользователя
- /replyme - Повторение текста отправки пользователя. (Команда для проверки работоспособности бота)
